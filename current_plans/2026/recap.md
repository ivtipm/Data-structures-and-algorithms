# Поисковая структура данных
Структура данных в который эффективно реализован поиск

Стоит использовать, когда операции поиска сильно более частые чем операции изменения (вставка, удаление)

Сложность указана для среднего случая

- Массив. 
  - Линейный Поиск O(n) 

- **Сортированный Массив**. 
  - Бинарный Поиск                       - O(log n)
  - Вставка [в правильную позицию]       - O(n)
  - Удаление [конкретного значения]      - O(n)

- Связный Список.
  - Линейный Поиск O(n) 

- **Skip-List**
  - Поиск                                - O (log n)
  - Вставка [в правильную позицию]       - O (log n)
  - Удаление [конкретного значения]      - O (log n)

# Алгоритмы поиска
- Линейный (последовательный) -- O(n)
- Бинарный
- Интерполяционный








# Очередь
- Дин. массив
  - **Добавить (в начало мас.) --    O(n)**   или (в конец мас.)   -- O(1)
  - **Удалить (из конца мас.) --     O(1)**   или (из начала мас) -- O(n)
  - Прочитать первый элемент -- O(1)
  - Получить размер          -- O(1)  
  - Доп память:              -- O(1)

- Связный список
  - **Добавить (в начало спис.) --    O(1)**   или (в конец спис.)   -- O(1)
  - **Удалить (из конца спис.) --     O(1)**   или (из начала спис.) -- O(1)
  - Прочитать первый элемент -- O(1)
  - Получить размер          -- O(1)
  - Доп память:              -- O(n) 


- **Кольцевой буффер (динимаческий)**
  - **Добавить (в начало КБ) --    O(1)**   или (в конец КБ)   -- O(1)
  - **Удалить (из конца КБ) --     O(1)**   или (из начала КБ) -- O(1)
  - Прочитать первый элемент -- O(1)
  - Получить размер          -- O(1)
  - Доп память:              -- O(1)


# Очередь Приоритетов
- Дин. сортированный массив
  - Добавить в нужную позицию      -- O( n )
    - Найти позицию -- O( log2 n )
    - Сдвинуть элементы для вставки -- O(n)
  - Удалить (из начала очереди)      -- O(1)  
     Реализовано как удаление из конца сорт. массива
  - Peek -- просмотр первого элемента в очереди -- O(1)
    Реализовано как просмотр последнего эл-та в массиве
  - size  -- O(1)
  - clear -- O(1) если не нужно вызывать деструкторы эл-в очереди
          -- O(n) если    нужно вызывать деструкторы эл-в очереди
